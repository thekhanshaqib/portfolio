# Shaqib Iqbal | Technical Product Manager Portfolio

A high-performance, interactive portfolio built with Next.js, React, Three.js, and Framer Motion. This portfolio showcases technical leadership, AI orchestration, and 0-1 product development experience.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + ShadCN UI
- **Animations:** Lucide React + Three.js
- **AI Integration:** Genkit + Google Gemini (AI Portfolio Assistant)

## Project Structure

- `/src/components/sections`: Individual UI sections (Hero, Journey, Skills, etc.)
- `/src/components/canvas`: Three.js rendering logic.
- `/src/ai`: Genkit flows and AI assistant configuration.

## How to Push to GitHub

To push this project to your own GitHub repository (`https://github.com/thekhanshaqib/portfolio`), follow these steps from your local terminal:

1. **Initialize Git (if not already done):**
   ```bash
   git init
   ```

2. **Add your remote repository:**
   ```bash
   git remote add origin https://github.com/thekhanshaqib/portfolio.git
   ```

3. **Stage and commit your files:**
   ```bash
   git add .
   git commit -m "Initial commit: Professional PM Portfolio with AI Assistant"
   ```

4. **Push to the main branch:**
   ```bash
   git branch -M main
   git push -u origin main
   ```

## Local Development

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
